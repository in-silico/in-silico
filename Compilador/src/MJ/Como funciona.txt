Para iniciar, y segun lo especificado en el documento
principal y la tercera entrega, se debe acceder desde
consola (ejecutar -> cmd en windows) y se debe ejecutar
el comando de la siguiente forma:

python mj.py nombreArchivo.mj -dump-ast -dump-symtab

Donde nombreArchivo es el nombre del archivo a analizar
y debe encontrarse en el directorio src.

Para una prueba simple, copiar el archivo bueno.mj de 
test\semantico\bueno.mj, a src\MJ\ y ejecutar el comando:

python mj.py bueno.mj -dump-ast -dump-symtab
